写一个脚本showlogged.sh，其用法格式为：
showlogged.sh -v -v -h|–help
其中，-h选项只能单独使用，业内关于显示帮助信息；-c选项时，显示当前期待能够上登录的所有用户数；如果同时使用了-v选项，则既显示同时登录的用户数，又显示登录的用户的相关信息。
