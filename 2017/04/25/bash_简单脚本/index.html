bash循环语句使用

字符测试：
[ $A = $B ] 等号两边都需要加空格，不然会被认为是附值。
== : 测试是否相等，相等为真，不等为假
!= : 测试是否不等，不等为真，相等为假

>  : 大于
<  : 小于
-n string ： 测试指定字符串是否为空，空则真，不空则假
-s string ： 测试指定字符串是否为不空，不空为真，空则为假

脚本练习：
传递一个用户名参数给脚本，判断此用户的用户名跟其基本组的组名是否一致，并将结果显示出来。

#!/bin/bash
#
if ! id $1 &>/dev/null; then
echo “No such user”
exit 10
fi

if [ ‘id -n -u $1’ == ‘id -n -g $1’ ]; then
echo “yiyang”
else
echo “buyiyang”
fi

传递一个参数（单个字符）给脚本，如参数为q、Q、quit、Quit，就退出脚本，否则，就显示用户的参数；

#!/bin/bash
#
if [ $1 = ‘q’ ];then
echo “Quiting…”
exit 1
elif [ $1 = ‘Q’ ];then
echo “Quiting…”
exit 2
elif [ $1 = ‘quit’ ];then
echo “Quiting…”
exit 3
elif [ #1 = ‘Quit’ ];then
echo “Quiting…”
exit 4
else
echo $1
fi
可以使用&&来进行判断

传递三个参数恶搞i脚本，第一个为整数，第二个为算出运算符，第三个为整数，将计算结果显示出来，要求保留两位精度。形如：
./calc.sh 5 / 2
提示：bc命令
echo “scale=2;111/22;” | bc(保留2位小数，111/2)
同样可以：
bc <<< “scale=2;111/22;”

传递3个参数给脚本，参数均为用户名。将此些用户的帐号信息提取出来后放置于/tmp/testusers.txt文件中，并要求每一行行首有行号。
echo “1 $LINE” >> /tmp/testusers
echo “2 $LINE” >> /tmp/testusers

脚本需求：
判断当前主机的CPU生产商，其信息在/proc/cpuinfo文件中vendor id一行中。
如果其生产商为AuthenticAMD,就显示其为AMD公司；
如果其生产商为GenuineIntel,就显示其为Intel公司；
否则，就说其为非主流CPU公司；

求1,..,100之和；

#!/bin/bash
#
declare -i SUM=0

for I in {1..100};do
let SUM=$[$SUM+$I]
done

echo “The sum is: $SUM”

添加10个用户user1到user10，密码同用户名；但要求只有用户不存在的情况下才能添加；

#!/bin/bash
#

for I in {1..10}; do
if id user$I &> /dev/null; then
echo “user$I exists”
else
useradd user$I
echo user$I | passwd –stdin user$I &> /dev/null
echo “add user user$I finished”
fi
done

删除10个用户

#!/bin/bash
#

for I in {1..10}; do
if id user$I &> /dev/null; then
userdel -r user$I
echo “delete user$I finished”
else
echo “user$I not exists”
fi
done

给定脚本参数，如果是–add则添加10个用户，如果是–del则删除10个用户

#!/bin/bash
#

if [ $# -lt 1 ]; then
echo “Usage:adminusers ARG”
exit 3
fi

if [ $1 == ‘–add’ ]; then
for I in {1..10]; do
if id user$I &> /dev/null; then
echo “user$I exists”
else
useradd user$I
echo user$I | passwd –stdin user$I &> /dev/null
echo “add user$I finished”
fi
done
elif [ $1 == ‘–del’ ]; then
for I in {1..10}; do
if id user$I &> /dev/null; then
userdel -r user$I
echo “delete user$I finished”
else
echo “no user$I”
fi
done
else
echo “Unknown ARG”
exit 4
fi

指定使用–add USER1，USER2，ABC，tom；如果不存在则添加用户USER1，USER2，ABC，tom；并显示添加成功；
同样使用–add USER1，USER2，ABC，tom；如果存在则删除用户USER1，USER2，ABC，tom；并显示删除成功；
同时给予–help帮助信息

#!/bin/bash
#

if [ $# -lt 1 ]; then
echo “Usage:yyuser ARG”
exit 3
fi

if [ $1 == ‘–add’ ]; then
for I in echo $2 | sed 's/,/ /g'; do
if id $I &> /dev/null; then
echo “$I exists”
else
useradd $I
echo $I | passwd –stdin $I &> /dev/null
echo “add $I finished”
fi
done
elif [ $1 == ‘–del’ ]; then
for I in echo $2 | sed 's/,/ /g'; do
if id $I &> /dev/null; then
userdel -r $I
echo “delete $I finished”
else
echo “$I not exist”
fi
done
elif [ $1 == ‘–help’ ]; then
echo “Usage: yyuser.sh –add USER1,USER2,… | –del USER1,USER2,… | –help”
else
echo “Unknown options”
fi
